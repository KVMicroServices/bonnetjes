# Bonnetjes v26.05.20.1

Analytics dashboard with metrics, volume charts, and audit log system.

**Branch**: feat/analytics → develop
**Date**: 2026-05-20
**Scope**: 49 files changed, 3143 insertions, 82 deletions

---

## Features

- Added admin analytics page with metrics overview (approval/rejection rates, status counts) and receipt volume chart with hourly/daily/weekly grouping
- Added audit log system with fire-and-forget event recording across all key integration points (OCR, moderation, settings, user management, dispute verification)
- Added audit log tab in analytics page with category filters, paginated table, and human-readable summaries
- Added user settings placeholder page accessible to all authenticated users
- Added Analytics and Admin navigation links in header for admin users

## Architecture

- Created `AuditLog` Prisma model with composite indexes for efficient category and time-based queries
- Created `analytics-service` with dependency-injected database access for metrics and volume data
- Created `audit-log-service` with fire-and-forget writer (errors logged internally, never propagated) and cursor-based pagination query
- Extended analytics API route with `type=audit` handler supporting category filtering and cursor pagination
- Renamed admin settings nav link to "Admin" with Shield icon, separated from user-facing "Settings" link
- Added `queuedAt` field to Receipt model for queue timing visibility
- Added unit tests for analytics-service and audit-log-service
